How it works

Powerful diff, zero friction

Built around a Myers O(ND) algorithm with Hunt-McIlroy LCS for large files — the same diff engine used by Git.

01
Split & unified views
Toggle between side-by-side split view and unified inline diff depending on your preference.
02
Smart diff options
Ignore whitespace, case, blank lines, or apply custom regex patterns to filter out noise.
03
Excel sheet mapping
Choose which sheets to compare, set a key column for row-level matching, detect formula changes.
04
Export results
Share your diff as HTML, JSON, CSV, or a patch file directly from the app.
05
Change navigation
Jump between changes with the built-in navigator. See a summary of additions, deletions, and edits.
06
Landscape mode
Full landscape support on all screens — more diff content, less chrome.
